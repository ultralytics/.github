# Ultralytics GitHub Community Resources

Welcome to the Ultralytics `.github` repository! This special repository serves as a central hub for our GitHub Actions workflows, issue and pull request templates, contributing guidelines, and other shared resources for the Ultralytics organization.

## Purpose of this Repository

The `.github` repository is a unique space that helps us manage and standardize our community and contribution practices across all our repositories. It contains:

- **GitHub Actions Workflows**: Predefined actions that help automate our software development practices.
- **Issue Templates**: Standard templates for submitting issues, which help maintain clarity and consistency in reporting bugs or requesting features.
- **Pull Request Templates**: Templates to guide contributors in providing necessary details when submitting pull requests.
- **Contributing Guidelines**: Information on how to contribute to Ultralytics projects, ensuring a smooth collaboration process.

## How to Use These Resources

- **For Ultralytics Team Members**: Use these templates and workflows as a starting point for new repositories and to maintain consistency in existing ones.
- **For Community Contributors**: Please adhere to the guidelines and templates provided when contributing to our projects. This helps us manage and review your contributions effectively.

## Feedback and Contributions

We love your input! Ultralytics open-source efforts would not be possible without help from our community. Please see our [Contributing Guide](https://docs.ultralytics.com/help/contributing) to get started, and fill out our [Survey](https://ultralytics.com/survey?utm_source=github&utm_medium=social&utm_campaign=Survey) to send us feedback on your experience.
